Quandl.search(query = Search Term, page = int, source = Specific source to search, silent = TRUE|FALSE)

query is the only mandatory argument and represents the term you wish to search for on Quandl. page is which page of search result you wish to return (default = 1), and source lets you choose the specific source you wish to search for. By putting silent to FALSE you only see the first 3 results printed to the console. If silent is set to TRUE it does not print anything to screen.

# Given the recent popularity of Bitcoins, you want to do a search for Bitcoin. You only want the first 3 results from the first page. You have no idea on what specific source to search for so no need to use the source parameter.
Quandl.search(query = "Bitcoin", silent = FALSE)
